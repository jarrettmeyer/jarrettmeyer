Scrollytelling is a neologism of scroll and storytelling. The concept is to tell a story with data as a user scrolls through a web site. Our picture starts simple, and we add more and more elements as the user reads through the data. Additionally, we make a heavy use of transitions, so changes are not jarring to the user.
